# 安装

进入项目目录后，使用以下方式进行安装。

# 本地引用

到官方仓库 下载 或使用命令行克隆到本地

$ git clone git@github.com:yuyuye958/GuLu-UI.git

下载 / 克隆到本地后，你就可以注册某个组件来使用了。

# NPM 安装

推荐使用npm / yarn来安装，享受生态圈和工具链带来的便利。

$ npm install vue-gulu-ui --save
$ yarn add vue-gulu-ui --save
