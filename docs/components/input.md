# Input 输入框

基础表单组件，支持双向绑定和原生事件。

## 基础用法

<ClientOnly>
  <input-demo1></input-demo1>
</ClientOnly>

## 双向绑定

<ClientOnly>
  <input-demo2></input-demo2>
</ClientOnly>

## Attributes

参数 | 说明 | 类型 | 可选值 | 默认值
:-:| - | :-: | :-: | :-: 
readonly | 只读状态 | Boolean | —— | false
disabled | 禁止输入状态 | Boolean | —— | false
error | 错误状态，提示错误信息 | String | —— | ——
