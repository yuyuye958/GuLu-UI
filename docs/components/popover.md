# Popover 弹出框

常用于展示提示信息。

## 基础用法

<ClientOnly>
  <popover-demo1></popover-demo1>
</ClientOnly>

## 四种方向

<ClientOnly>
  <popover-demo2></popover-demo2>
</ClientOnly>

## Attributes

参数 | 说明 | 类型 | 可选值 | 默认值
:-:| - | :-: | :-: | :-: 
position | 弹出框的位置，默认顶部 | String | top, right, bottom, left | top
trigger | 弹出框的触发方式，默认点击触发 | String | click, hover | click
